# libs

В эту папку помещаются все дополнительные deb‑пакеты библиотек (зависимостей), которые вы скачиваете с портала ИТС 1С вместе с основными пакетами PostgreSQL для вашей версии и операционной системы (чаще всего — для Debian 12).

Возможные примеры:
```

libicu72_*.deb
libreadline8_*.deb
libldap-2.5-0_*.deb
krb5-user_*.deb
libcurl4_*.deb
libxml2_*.deb

```

Все пакеты должны быть скачаны с ИТС 1C и соответствовать версии основной поставки!
