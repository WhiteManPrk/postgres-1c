# core

В эту папку помещаются deb‑пакеты PostgreSQL, загруженные из личного кабинета ИТС 1С для нужной версии.

Сюда нужно скопировать:
- postgresql-<версия>_*.deb — сервер
- postgresql-client-<версия>_*.deb — клиентские утилиты
- libpq5_*.deb — клиентская библиотека

Пример (для версии 16):
```

postgresql-16_16.9-*.deb
postgresql-client-16_16.9-*.deb
libpq5_16.9-*.deb

```

Все файлы берутся только из официального дистрибутива на портале поддержки 1C (ИТС)!
